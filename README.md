# Customer-Churn-Dashboard
Developed an interactive customer churn analysis dashboard using Power BI to identify key factors affecting customer retention in a telecom dataset. Performed data cleaning and transformation using Power Query, handling missing values and improving data consistency. Created KPIs such as Churn Rate, Total Customers, Active Customers, Monthly Charges, and Contract Type Distribution. Analyzed churn behavior based on payment method, contract type, tenure, internet service, gender, and billing preferences. Designed multiple visuals including bar charts, slicers, filters, and trend analysis dashboards to support business decision making. Identify high risk churn segments such as month-to-month contract users and electronic check customers, suggesting retention strategies. Implemented interactive bookmarks and navigation buttons for better dashboard & storytelling.
